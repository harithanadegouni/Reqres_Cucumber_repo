Setup:
1. Create maven project
2. Add dependencies in pom.xml
3. Create features files, Stepdefinitions as per the test case steps
4. Create Test runner class and add correct cucumber options

Run the test case:
Right click on TestRunner class and click on Run Test runner option

Reports:
After execution of the scenarios reports will be generated based on the options cucumber options (ex - html report, xml report, json report)
Open html file through browser and verify test outcome
